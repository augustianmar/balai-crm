# BALAI Customer Relationship Management

A first static prototype for the BALAI CRM interface. It is designed to resemble the provided CRM reference: dark left navigation, people list, selected contact workspace, quick action toolbar, activity cards, and contact info panel.

## Open

Open `balai-crm-standalone.html` in a browser.

To open it from a launcher page, open `open-crm.html` and click **Open CRM in new tab**.

## Check

```sh
npm run check
```

The app has no runtime dependencies. The check command only verifies the JavaScript syntax with Node.
